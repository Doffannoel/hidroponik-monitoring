# Tikus Kota - Landing Page

Starter project Next.js App Router + TypeScript + Tailwind CSS 3 untuk web monitoring tanaman hidroponik.

## Halaman yang sudah disiapkan
- Landing Page (`/`)
- Panduan (`/panduan`)
- Sign Up (`/signup`)
- Sign In (`/signin`)
- Dashboard (`/dashboard`)
- Detail Tanaman (`/detail-tanaman`)
- List Tanaman (`/list-tanaman`)

## Jalankan proyek
```bash
npm install
npm run dev
```

## Struktur modular
- `app/(public)` untuk route halaman
- `components/landing` untuk section landing page
- `components/layout` untuk header/footer
- `components/shared` untuk komponen reusable
- `data` untuk data statis section
- `lib/constants` untuk konstanta tema
